## 📌 Featured Projects

### 🏥 Public-Health Analytics Platform *(DEA Lab)*
A full-stack platform on **NFHS-5 national survey microdata**, computing 14 survey-weighted malnutrition indicators.
- Incremental, idempotent **ETL framework** (S3 ETag + hash manifests) that cut re-run time from hours to seconds
- ML models predicting **child anthropometric failure (CIAF) risk**, served via a real-time FastAPI API
- Production **React 18 + Vite dashboard** with D3.js/Recharts charts and Leaflet choropleth maps
- Full **CI/CD** (lint, Vitest, Playwright, Lighthouse) deploying to Vercel & Render

`Python` `AWS S3` `FastAPI` `React` `D3.js` `GitHub Actions`

### 📉 Patient Churn Prediction — Healthcare Retention Analytics
ML model on 1,200+ patient records predicting churn with **85% accuracy**, cutting manual analysis time by 75%.

`Python` `Machine Learning` `Docker` `FastAPI` `Streamlit` `CI/CD`

### 🌿 Prakriti.AI — Clinical Assessment & Recommendation System
A rule-based clinical engine using 17 diagnostic factors and 32 parameters, with a responsive React interface — reducing manual assessment effort by ~65–70%.

`Python` `Rule-Based Logic` `React` `JavaScript`

### 💰 Insurance Charges Prediction
Linear Regression model (R² = 0.72–0.75) deployed as an interactive Streamlit app for real-time premium estimation.

`Python` `Machine Learning` `Streamlit`
